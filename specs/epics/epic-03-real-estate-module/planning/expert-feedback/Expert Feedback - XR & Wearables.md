### Phase 1: Basic XR Support (Sprint 03 or Post-Launch)

**Supported Devices** (prioritize by real estate adoption):
1. Meta Ray-Ban Smart Glasses (most affordable, widest adoption)
2. Apple Vision Pro (high-end commercial, luxury agents)
3. Vuzix Shield (property inspectors, investors)

**Integration Approach**:

#### Option A: Cloud Upload (Easiest - MVP)
```
XR Glasses ‚Üí Meta/Apple Cloud ‚Üí Transcript Parser API ‚Üí IndexedDB

Flow:
1. Agent records showing on Ray-Ban glasses
2. Video/audio uploads to Meta cloud (automatic via Meta app)
3. Meta app on phone has "Share to Transcript Parser" option
4. Transcript Parser API downloads from Meta cloud
5. Processes (transcription, AI features)
6. Stores locally in IndexedDB (offline-capable after initial sync)
```

**Pros**:
- No custom hardware integration needed
- Uses existing cloud infrastructure (Meta, Apple)
- Simple user flow (agent already using Meta app)

**Cons**:
- Privacy concern (data goes through Meta/Apple) - David won't use this
- Depends on third-party APIs (may break)
- Storage limits on glasses (32GB = ~8 hours video)

**Best for**: Lisa, Carmen, Michael (residential/property management - less NDA concerns)

---

#### Option B: Direct Device Integration (Better Privacy - Future)
```
XR Glasses ‚Üí Bluetooth/Wi-Fi Direct ‚Üí Phone App ‚Üí Transcript Parser ‚Üí IndexedDB

Flow:
1. Agent wears glasses, starts recording
2. Glasses stream video/audio to phone app (Bluetooth or Wi-Fi Direct)
3. Transcript Parser mobile app receives stream
4. Real-time transcription (or buffer for later processing)
5. Stores locally (no cloud intermediary)
```

**Pros**:
- Privacy-first (no cloud upload) - David can use for NDA-compliant recordings
- Faster (no cloud round-trip)
- Works offline (airplane mode, poor cell signal)

**Cons**:
- Requires custom mobile app development (8-12 story points)
- Bluetooth bandwidth limits (compressed video)
- Battery drain on phone (receiving stream)

**Best for**: David (commercial broker with NDAs), privacy-conscious agents

Default: Cloud upload (easy onboarding - 80% of users)

## üîê Privacy & Ethics for Real Estate XR Recording

### Challenge: Recording Laws & Client Consent

**Issues**:
1. **State recording laws**:
   - Two-party consent states (California, Florida, etc.) - BOTH parties must consent
   - One-party consent states (Texas, New York, etc.) - Only one party needs to consent
   - Real estate agents must comply with state law

2. **Client privacy**:
   - Recording client financial info (pre-approval amount, down payment)
   - Personal details (family situation, employment, health)
   - Property concerns (divorce, foreclosure, urgent sale)

3. **Fair Housing compliance**:
   - Recording protects against Fair Housing complaints
   - But must not capture protected class information (race, religion, family status)
   - Accurate records crucial for legal defense

---
